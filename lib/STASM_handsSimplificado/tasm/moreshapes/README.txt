This directory contains additional shape files.  The files are
separated to reduce the size of the Stasm zip file.

These files are not needed for testing Stasm or for the examples in
the Tasm User Manual.

If this directory contains no shape files, please download them
from the "moretrain" zip file on the Stasm web page.

As of Dec 24, 2013, the additional files are:
    ar.shape
    bioid.shape
    muct68.shape
    muct76.shape
    muct77.shape
    put199.shape
    xm2vts.shape
